# aws-cloud-native-architecture-topic3

## AWS Cloud Native Architecture Assignment

This repository contains the Cloud (AWS) Hands-on Topic 3 assessment project focused on designing and implementing a cloud-native architecture using AWS services.

## Services Used

- Amazon EC2
- Elastic Load Balancer
- Auto Scaling
- Amazon S3
- Amazon VPC
- Security Groups
- IAM

## Topics Covered

- Cloud-native architecture
- Scalable application deployment
- Distributed cloud infrastructure
- Load balancing
- Auto scaling configuration
- Cloud networking
- Secure resource management

## Steps Performed

1. Created cloud infrastructure resources in AWS.
2. Configured networking and security settings.
3. Launched EC2 instances for application hosting.
4. Configured Load Balancer and Auto Scaling.
5. Tested scalability and resource communication.
6. Verified successful deployment and accessibility.
7. Uploaded project documentation and screenshots to GitHub.

## Challenges Faced

- Service integration issues
- Auto Scaling configuration errors
- Security group and networking problems
- Connectivity issues between resources

## Outcome

Successfully completed the AWS Cloud Native Architecture assessment by implementing scalable and reliable cloud infrastructure components.
